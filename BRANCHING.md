# Branching Strategy

- `main`: production-ready stable branch
- `dev`: integration branch
- `feature/<scope>-<name>`: feature work branch
- `fix/<scope>-<name>`: bug fix branch

Examples:
- `feature/backend-upload-api`
- `feature/frontend-3d-editor`
- `fix/backend-upload-timeout`

Commit prefix:
- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
